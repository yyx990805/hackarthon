# Konfiguracja

W celu wygodnego przeglądania zbiorów warto dokonfigurować przeglądarkę,
by automatycznie dodawała nagłówki autoryzujące
oraz w czytelniejszy sposób.

1. Instalacja rozszerzenia Modify Header Value (HTTP Headers)

<img src="./Zrzut ekranu 2022-09-21 o 15.57.16 modify header.png"/>

2. Następnie uzupełniamy tabelkę

```
Domain: https://apihackaton.zacheta.art.pl/*
Header name: Accept
Header value: application/vnd.api+json

Domain
https://apihackaton.zacheta.art.pl/*
Header name: Authorization
Header value: Bearer 21a06898069e580(...256 znaków, token otrzymany na maila)
```

<img src="./Zrzut ekranu 2022-09-21 o 15.57.16 modify header.png"/>
