# System inwentaryzacji zbiorów

Aplikacja łączy w sobie arkusz kalkulacyjny, wczytywanie danych z API oraz wyświetlanie szczegółów po wybraniu.

## Budowa

Layout oparty na bibliotece tailwind flex.

```js
export const Layout = ({children, color}) => (
  <div>
    <div>menu</div>
    <div>
      <div>arkusz</div>
      <div>items details</div>
    </div>
  </div>
);
<Layout/>
```

export const Spreadsheet = ({children, color}) => {
  return (<table>
    {[1,2,3,4].map(e => <tr>{[1,2,3,4].map(e=><td>{e}</td>)}</tr> )}
  </table>)
}

export const Layout = ({children, color}) => {
  const [isSpreadsheet, setIsSpreadsheet] = React.useState(false)
  return (<div className="flex border-solid border-2">
    <div>menu</div>
    <div>
      {!isSpreadsheet && <div>arkusz <button onClick={() => setIsSpreadsheet(true)}>zastąp arkuszem</button></div>}
      {isSpreadsheet && <div><Spreadsheet/></div>}
      <div>szczegóły*<br/><br/><small>* okienko pojawi się dopiero po zaznaczeniu wiersza</small></div>
    </div>
  </div>);
}

<Layout/>
